# Prompt 3: Creative Writing (Penulisan Kreatif)

## Deskripsi
Prompt ini digunakan untuk menghasilkan konten kreatif seperti cerita pendek, puisi, atau deskripsi produk
berdasarkan parameter yang diberikan oleh pengguna.

## Template Prompt

```
Kamu adalah penulis kreatif yang berbakat.

Tulis sebuah {{jenis_konten}} dengan ketentuan berikut:
- Tema     : {{tema}}
- Gaya     : {{gaya_penulisan}}
- Panjang  : {{panjang}} kata
- Audiens  : {{target_audiens}}

Informasi tambahan (opsional):
{{informasi_tambahan}}

Pastikan tulisan:
1. Menarik dan mengalir dengan baik.
2. Sesuai dengan gaya dan audiens yang ditentukan.
3. Orisinal dan bebas dari klise berlebihan.

Mulai menulis sekarang:
```

## Contoh Penggunaan

**Input:**
```
Jenis konten  : cerita pendek
Tema          : persahabatan di era digital
Gaya          : ringan, humoris, modern
Panjang       : 150 kata
Audiens       : remaja usia 15-20 tahun
Informasi tambahan: tokoh utama bernama Raka, seorang gamer yang susah bergaul di dunia nyata
```

**Output yang Diharapkan:**
```
Raka lebih hafal jadwal server maintenance game kesayangannya daripada jadwal kelasnya sendiri.
Dunia nyata? Membingungkan. Dunia digital? Rumahnya.

Sampai suatu malam, seorang pemain baru masuk ke guild-nya dengan username "NoobSlayer99".
Ternyata dia teman sekelas Raka yang selama ini hanya dikenal sebagai "si anak duduk di pojok".

"Lo yang selalu kasih support di boss fight kemarin?" tanya NoobSlayer99 lewat chat.
"Iya," jawab Raka singkat, jarinya gemetar.

Keesokan harinya di kelas, mereka saling pandang. Canggung. Lalu NoobSlayer99—atau Dika,
ternyata namanya—berbisik, "GG kemarin."

Raka tersenyum. Mungkin dunia nyata tidak terlalu berbeda dengan dunia digital.
Kamu hanya perlu menemukan guild yang tepat.
```

## Tips
- Isi semua parameter `{{...}}` sesuai kebutuhan Anda.
- `{{jenis_konten}}` bisa diisi: cerita pendek, puisi, slogan, deskripsi produk, caption media sosial, dll.
- `{{gaya_penulisan}}` contohnya: formal, santai, puitis, teknis, persuasif, naratif.
- `{{target_audiens}}` contohnya: anak-anak, profesional, ibu rumah tangga, mahasiswa, dll.
- Untuk hasil terbaik, semakin detail informasi tambahan yang diberikan, semakin relevan output yang dihasilkan.
