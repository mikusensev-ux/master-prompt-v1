# master-prompt-v1
Mastering the prompt engineering v1

## 📁 Struktur Repositori

```
prompts/
└── basic/
    ├── 01-summarization.md       # Prompt untuk meringkas teks
    ├── 02-question-answering.md  # Prompt untuk tanya jawab berbasis konteks
    └── 03-creative-writing.md   # Prompt untuk penulisan kreatif
```

## 🚀 Prompt Basic

Koleksi 3 prompt dasar yang sering digunakan dalam prompt engineering:

| No | Nama | Deskripsi | File |
|----|------|-----------|------|
| 1 | **Summarization** | Meringkas teks panjang menjadi poin-poin utama | [01-summarization.md](prompts/basic/01-summarization.md) |
| 2 | **Question Answering** | Menjawab pertanyaan berdasarkan konteks yang diberikan | [02-question-answering.md](prompts/basic/02-question-answering.md) |
| 3 | **Creative Writing** | Menghasilkan konten kreatif (cerita, puisi, caption, dll) | [03-creative-writing.md](prompts/basic/03-creative-writing.md) |

## 💡 Cara Menggunakan

Setiap file prompt berisi:
- **Deskripsi** – penjelasan fungsi prompt
- **Template Prompt** – template siap pakai dengan placeholder `{{variabel}}`
- **Contoh Penggunaan** – contoh input dan output yang diharapkan
- **Tips** – panduan untuk menyesuaikan prompt sesuai kebutuhan
